在這章節認識的CSS語法

  1. 文字顏色:color
  2. 文字字形:font-family
  3. 文字邊框:boder
  4. 文字底線:test-decoration
  5. 文字行距:line-height
  6. 其他

下列的哪個emmet語法是錯誤的?

  • ! = HTML5環境
  • link:CSS = 匯入CSS
  • P*4 = 顯示四個標籤
  • h1 = 展開h1的開頭與結尾html標籤
  • lorem = 自動幫我們寫完網頁
