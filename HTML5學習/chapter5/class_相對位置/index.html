position絕對位置應用

  • 相較float，更加活用
  • 利用position css語法去下參數，參數分為以下幾種
  • 絕對定位可以讓元素與元素之間重疊再一起

1.static：屬性默認值為:static，即該元素出現在文檔的常規位置，不會重新定位。通常此屬性值可以不設置，除非是要覆蓋之前的定義。

2.relative：表現的和 static 一樣，除非你增加了一些額外的屬性。他是相對定位

在元素內設定 top 、 right 、 bottom 和 left 屬性，會使其元素「相對地」調整其原本該出現的所在位置，而不管這些「相對定位」過的元素如何在頁面上移動位置或增加了多少空間，都不會影響到原本其他元素所在的位置。

4.fixed：元素會相對於瀏覽器視窗來定位，這意味著即便頁面捲動，它還是會固定在相同的位置。

3.absolute:絕對定位，便可以將元素放在文檔中任何想放的位置。
