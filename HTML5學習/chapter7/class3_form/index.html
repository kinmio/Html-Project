表格練習

利用form語法設計表單，表單之內容會透過表單抓取前端資料

  • action:用來指定一個位址 (URL)，這個位址是告訴瀏覽器 (browser) 當 user 按送出表單後，要將表格的內容送去哪邊
  • type="txet">定義為文字
  • type="password"> 建立一個密碼文字輸入欄位，和 text 的差別是，使用者輸入的內容不會被明碼顯示在螢幕畫面中
  • type="checkbox"> 建立一個多選核取方塊
  • type="radio"> 建立一個單選選項按鈕
  • type="submit"> 建立一個送出表單的按鈕
  • 部分 type 都可以再搭配上 value 屬性，指定一個預設值。像是 type="text" 搭配 value= 指定輸入框中的預填文字。

進階用法

  • placeholder:該屬性可在文字框中顯示提示訊息
  • label:給表單的控制元件一個說明標題，通常for會對應id，而id的命名會與name一樣。
